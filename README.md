## 🗺️ My Technical Roadmap

Following a rigorous, sequential training index, I am systematically auditing, coding, and proving proficiency across these core structural milestones:

### 🌟 Part 1: Linear Matrix & Windows

1. Arrays + Prefix Sum + Difference Array (Static contiguous storage, 1D/2D prefix range accumulation, O(1) interval updates)
2. Hashing (Key-value maps & sets, collision mechanics, frequency trackers)
3. Two Pointers (Converging bound searches, diverging centers, fast & slow velocity tracking)
4. Sliding Window (Fixed window constraints, variable subarray expansion/contraction)
5. Binary Search (Sorted space division, monotonic search on answers)

### ⛓️ Part 2: Linear Data & Call Stacks

6. Linked List (Singly, doubly, & circular node chains, pointer flipping & rewiring)
7. Stack & Queue (LIFO/FIFO control structures, monotonic boundary searching, deque sliding window optimization)
8. Recursion (System call stack frames, divide & conquer frameworks)
9. Bit Manipulation (Bitwise logic operators &, |, ^, ~, custom mask configurations)

### 🌲 Part 3: Graphs & Hierarchies

10. Trees & BST (Binary tree traversals, binary search tree properties, self-balancing tree invariants)
11. Heap / Priority Queue (Array coordinates mapping, sift-up & sift-down balance routines)
12. Backtracking (Decision tree space exploration, the "Undo" state rollback mechanics)
13. Graphs (Adjacency matrices vs. adjacency lists, BFS layer & DFS deep stack invasions, topological sorting & shortest path frameworks)

### 📈 Part 4: Strategic Optimization

14. Greedy (Local choice optimization bounds, proof-of-correctness logic)
15. Dynamic Programming (Overlapping subproblem memoization, top-down caching vs. bottom-up table loops)

### ⚡ Part 5: Hard-Core Advanced Structures

16. Trie (Digital prefix search trees, alphabetical character chains)
17. DSU (Union Find) (Disjoint set unions, path compression & union by rank/size)
18. Segment Tree / Fenwick Tree (Interval divide-and-conquer ranges, lazy propagation updates, least significant bit array tracking)

### 🚀 Part 6: Execution

19. Contests + Revision (Live competitive arena coding, spaced repetition problem auditing)
